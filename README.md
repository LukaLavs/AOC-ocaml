# Advent of Code 2025 (Ocaml)

## Instructions 

```
dune build
make run DAY=0 YEAR=2025
make run DAY=1 YEAR=2025
...
make run DAY=11 YEAR=2025
```

## Note

Days are zero indexed. Solutions are found in `src/solvers/year_2025`.
